PR (Point Region) Quadtree visualization

basics of PRQuadtrees:
-nodes describe regions
-regions are containers for 2dimensional data
-regions have 3 states: empty (no value, no children), leaf (has value, no children), parent (no value, has 4 children)
-if a point is inserted into a region that contains a point already (leaf), it will recursively split the region in four quarters until each point has its own region
-perform best with uniform point distribution (high point to depth ratio), performance not dependent on order of insertion
